#七日杀Mod启动器
## 这是什么？

这是一个为《七日杀》玩家设计的 Mod 启动器工具，支持：
- 一键切换游戏平台（Steam / Game Pass）
- Mod 启用/禁用（跨分区移动，不占双倍空间）
- 游戏目录自动定位
- Mod 网站快速跳转
-支持英语

## 如何使用

1. 下载 Releases 中的 `七日杀启动器 Setup.exe`
2. 安装后启动，点击“设置”选择游戏目录
3. 点击“扫描”自动定位，或手动选目录
4. 在 Mod 列表点击“禁用”即可将 Mod 移入隔离区
